# Superstore Sales Dashboard

## Overview
Interactive Excel dashboard analyzing Superstore sales data to uncover trends, KPIs, and business insights.

## Dataset
- **Source**: [Kaggle Superstore Dataset](https://www.kaggle.com/datasets/vivek468/superstore-dataset-final)
- **Rows/Columns**: ~9,994 rows, 21 columns

## Dashboard Features
- **KPIs**: Total Sales, Profit, Margin %, Order Count, Avg Discount
- **Visuals**: Sales by Category, Time Trend, Regional Performance, Top Products by Profit
- **Interactive Slicers**: Region, Category, Segment, Date (Timeline)

## Tools Used
- Microsoft Excel (Power Query for cleaning, PivotTables/Charts for visuals, Slicers for interactivity)
